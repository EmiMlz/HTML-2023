Java Scrypt

Conocido como JS (JavaScrypt), es un lenguaje enfocado en darle interactividad a las paginas(dinamicas, funcionalidades). Hoy en dia no solo se usa JS, ahora se usan frameworks como Angular, React o Vue. Caracteristicas:

  • Scripting lenguage.
  • Un lenguaje de programacion liviano de alto nivel.
  • Es un lenguaje interpretado.
  • Es un conjunto de lineas que puede usar cualquiera.
Se utiliza adentro de html con la etiqueta <script> El java scrypt se mete dentro del Head cuando se quiere capturar eventos, por ejemplo cuando se carga una pagina. Se utiliza dento del Body cuando es codigo que se ejecuta cuando la pagina es cargada. En codigo externo dentro de un .js, <noscript> cuando el usuario lo habilita Para asignar variables se usa el var x, el ; no es necesario.
