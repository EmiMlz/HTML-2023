CSS = Hojas de estilo en cascada.

Tiene tres tipos de caminos diferentes:

Hoja externa Archivo CSS que se vincula al documento HTML a través del elemento <link> y que define los estilos para todas las páginas del sitio web.
Hoja interna Archivo CSS que se incluye directamente en el documento HTML a través del elemento <style> y que define los estilos para esa página en particular.
Estilo en línea Estilo CSS que se aplica directamente a un elemento HTML a través del atributo "style".

Los estilos en línea tienen la prioridad más alta y anulan cualquier otro estilo definido a continuación.

En segundo lugar, se usan los estilos definidos en hojas de estilo internas (dentro de etiquetas <style> en la sección <head> del documento).

Por último, se toman los estilos definidos en un archivo de hojas de estilo externas.

Selector universal * y selectores individuales con palabra definida de las etiquetas (por ejemplo p o h1).

Padding: El padding (relleno) es el espacio entre el borde del elemento y su contenido. Se puede ajustar el padding con CSS para crear más o menos espacio entre el contenido y el borde. Cuando se establece padding en un elemento, está añadiendo espacio en el interior de ese elemento. Margin: El margin (margen) es el espacio entre el borde del elemento y los elementos circundantes. Puede ajustar los márgenes con CSS para crear más o menos espacio entre los elementos. Cuando se establece un margen en un elemento, está añadiendo espacio alrededor de ese elemento. Border: El border (borde) es la línea que rodea el elemento. Se puede personalizar el grosor, el estilo y el color del borde mediante CSS. Los div sirven para separar elementos en bloques, se pueden modificar uno a uno por su id. Span es igual que el div pero que no te salta de linea.
